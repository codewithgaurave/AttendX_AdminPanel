# AttendanceX - Admin Portal

Admin portal for attendance management, QR scanning, and employee management.

## Features
- QR Code scanning for attendance
- Employee management
- Office management
- Attendance logs and reports
- Salary calculations
- Holiday management
- Office-wise attendance tracking

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment
This app should be deployed to `admin.attendx.com` subdomain.

## Role Access
- Only Admin role users can access this portal
- Redirects to login if unauthorized
